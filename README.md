# JDBC
This repository contains the Java programs I practiced while learning JDBC (Java Database Connectivity).
The purpose of this practice is to understand how Java interacts with databases using JDBC APIs.
Topics I practised

Connecting Java to a database using JDBC
Loading JDBC drivers
Creating database connections
Executing SQL queries (SELECT, INSERT, UPDATE, DELETE)
Using Statement and PreparedStatement
Handling ResultSet
Closing database resources properly
Basic CRUD operations
